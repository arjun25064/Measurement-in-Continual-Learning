## Presentation materials from IIIT Delhi on continual learning systems for edge video analytics.

## Contents
### Ekya (Ekya_Arjun_Prasad.pdf) 
A review of the Ekya system (Bhardwaj et al., USENIX NSDI 2022), which addresses continuous learning of video analytics models on edge compute servers. Covers the core problem of balancing inference and retraining under tight GPU constraints, the Thief Scheduler, micro-profiling, and resource allocation strategies.
## Paper link - [ekya](https://arxiv.org/pdf/2012.10557)

### RECL (RECL_Presentation.pdf)
A presentation on Responsive Resource-Efficient Continuous Learning, which extends the ideas in Ekya by introducing a shared Model Zoo, lightweight gating-based model selection, and a progress-aware training scheduler. RECL reduces retraining compute by up to 3.2× while improving accuracy over baselines like Ekya and AMS.
